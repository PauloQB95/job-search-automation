# Job Search Project

## Descripción

Este proyecto automatiza la búsqueda de ofertas laborales en organismos internacionales y entidades públicas, con énfasis en puestos relacionados con políticas públicas, economía, análisis de datos e infraestructura.

## Tecnologías utilizadas

- Python
- Requests
- Pandas
- OpenPyXL

## Estado del proyecto

✅ Descarga de vacantes del BID

⬜ Banco Mundial

⬜ CAF

⬜ Naciones Unidas
